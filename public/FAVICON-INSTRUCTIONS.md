# Instruções para Favicons - Qualitec Instrumentos

## Arquivos Necessários

1. **favicon.svg** ✅ (já criado)
   - Favicon moderno em SVG
   - Suporte a modo escuro/claro
   - Escalável para qualquer tamanho

2. **favicon.ico** (pendente)
   - Converter o favicon.svg para ICO
   - Tamanhos: 16x16, 32x32, 48x48
   - Usar: https://favicon.io/favicon-converter/

3. **apple-touch-icon.png** (pendente)
   - Converter o favicon.svg para PNG 180x180
   - Bordas arredondadas para iOS
   - Usar: https://realfavicongenerator.net/

## Design Qualitec

### Cores Corporativas
- Azul Principal: #1e40af
- Azul Escuro: #1e3a8a  
- Azul Médio: #3b82f6
- Verde Indicador: #10b981
- Branco: #ffffff

### Elementos Visuais
- Logo "Q" estilizado
- Padrão industrial (círculos, linhas)
- Gradiente azul corporativo
- Elementos geométricos técnicos

## Como Gerar

1. Acesse: https://realfavicongenerator.net/
2. Faça upload do favicon.svg
3. Configure:
   - iOS: Manter design, bordas arredondadas
   - Android: Tema azul #1e40af
   - Windows: Fundo azul sólido
   - Safari: Cor #1e40af
4. Baixe e substitua os arquivos

## Verificação

Após gerar os favicons, teste em:
- Chrome/Edge (favicon.ico)
- Safari (apple-touch-icon.png)  
- Firefox (favicon.svg)
- Dispositivos móveis (PWA icons)
